## Контекст
Сеть дарксторов нуждалась в системе, которая могла бы в реальном времени пересчитывать маршруты, учитывать задержки и прогнозировать пиковые нагрузки.

## Вклад
- Разработал микросервисную архитектуру с использованием NestJS и Kafka для событийного взаимодействия.
- Построил GraphQL API c схемой Federation для интеграции с существующей CRM и приложением курьеров.
- Настроил витрину данных на ClickHouse для аналитики и алертинга, автоматизировал деплой через GitHub Actions.

## Эффект
Доставка заказов в пиковые часы ускорилась на 26%, сокращено количество отмен за счёт предсказания и перераспределения курьеров. Система выдерживает до 5k событий в секунду без деградации SLA.
