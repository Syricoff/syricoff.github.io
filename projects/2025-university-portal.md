## Задача
Создать веб-портал для учета посещаемости студентов с возможностью управления получением данных для администраторов, старост групп и студентов.

## Что сделал
- Спроектировал архитектуру приложения с разделением на фронтенд и бэкенд.
- Разработал RESTful API на FastAPI для управления данными о посещаемости.
- Создал фронтенд на React с удобным интерфейсом для разных ролей пользователей.
- Настроил базу данных PostgreSQL для хранения информации о студентах, группах и посещаемости.


## Результат
Портал успешно внедрен в учебный процесс, что позволило автоматизировать учет посещаемости и упростить взаимодействие между администрацией, старостами и студентами.
